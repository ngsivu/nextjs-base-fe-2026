# Base FE Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Dựng base frontend Next.js dùng lại được cho dự án mới, trung tính ở tầng backend, kèm một trang demo chứng minh mọi mảnh ghép hoạt động cùng nhau.

**Architecture:** Next.js App Router với cấu trúc feature-based. Mọi request HTTP đi qua một instance `ky` duy nhất trong `lib/http.ts`. State chia ba tầng: TanStack Query cho server state, nuqs cho URL state, Zustand cho client state (`appStore` trong bộ nhớ, `localStore` persist xuống localStorage). Backend chưa xác định nên dùng MSW mock toàn bộ API.

**Tech Stack:** Next.js 16.2.10, React 19.2.7, TypeScript 5.9.3, Tailwind CSS 4.3.3, shadcn/ui, TanStack Query 5, Zustand 5, nuqs 2, ky 2, Zod 4, React Hook Form 7, Vitest 4, Playwright 1.61, MSW 2.

**Spec:** `docs/superpowers/specs/2026-07-20-base-fe-design.md`

## Global Constraints

Áp dụng cho **mọi task**:

- **Package manager: pnpm.** Không dùng npm/yarn để cài dependency (trừ `npx`/`pnpm dlx` chạy CLI một lần).
- **Version pin chính xác:** `next@16.2.10`, `react@19.2.7`, `typescript@5.9.3`, `tailwindcss@4.3.3`. Không nâng TypeScript lên 6.x hay 7.x — `typescript-eslint@8` khai báo peer `typescript >=4.8.4 <6.1.0`, và ta cố ý giữ dư địa.
- **Quy ước đặt tên file:**
    - React component → **PascalCase** (`UserTable.tsx`, `AppInit.tsx`)
    - Hook → **camelCase** (`useDebounce.ts`)
    - Store, lib, util, config → **camelCase** (`appStore.ts`, `queryClient.ts`)
    - Route file của Next → theo Next (`page.tsx`, `layout.tsx`, `error.tsx`)
    - **Ngoại lệ:** `src/components/ui/` giữ kebab-case vì shadcn CLI sinh ra. Không đổi tên, không sửa file trong đó.
- **Không import chéo giữa các feature.** `features/a` không được import từ `features/b`. Dùng chung thì đẩy lên `lib/`, `hooks/`, `components/`.
- **Toast dùng `sonner`**, không dùng component `toast` cũ của shadcn (đã deprecated).
- **KHÔNG chạy `git commit`.** Theo quy tắc trong `CLAUDE.md` của user: mỗi task kết thúc bằng `git add` và in ra commit message gợi ý. User tự review và commit.
- **Không thêm** auth, ORM, i18n, orval, next-safe-action, Recharts, Sentry, Storybook, Dockerfile, CI/CD. Đây là ranh giới phạm vi đã chốt trong spec.

```tsx
'use client';

import { Moon, Sun } from 'lucide-react';
import { useTheme } from 'next-themes';
import { Button } from '@/components/ui/button';

export function ThemeToggle() {
	const { resolvedTheme, setTheme } = useTheme();

	return (
		<Button
			variant="ghost"
			size="icon"
			aria-label="Chuyển chế độ sáng tối"
			onClick={() => setTheme(resolvedTheme === 'dark' ? 'light' : 'dark')}
		>
			<Sun className="size-4 dark:hidden" />
			<Moon className="size-4 hidden dark:block" />
		</Button>
	);
}
```

Dùng CSS (`dark:hidden` / `dark:block`) để đổi icon thay vì đọc `resolvedTheme` lúc render — tránh hydration mismatch ở lần render đầu khi server chưa biết theme của client.

const DEFAULT_MESSAGE = 'Đã có lỗi xảy ra, vui lòng thử lại.';

const CODE_BY_STATUS: Record<number, string> = {
	400: 'BAD_REQUEST',
	401: 'UNAUTHORIZED',
	403: 'FORBIDDEN',
	404: 'NOT_FOUND',
	409: 'CONFLICT',
	422: 'VALIDATION_FAILED',
	429: 'TOO_MANY_REQUESTS',
};

export class ApiError extends Error {
	readonly status: number;
	readonly code: string;
	readonly details?: unknown;

	constructor(status: number, code: string, message: string, details?: unknown) {
		super(message);
		this.name = 'ApiError';
		this.status = status;
		this.code = code;
		this.details = details;
	}
}

function defaultCode(status: number): string {
	if (CODE_BY_STATUS[status]) return CODE_BY_STATUS[status];
	if (status >= 500) return 'SERVER_ERROR';
	return 'UNKNOWN';
}

export function toApiError(status: number, body: unknown): ApiError {
	const payload = typeof body === 'object' && body !== null ? (body as Record<string, unknown>) : {};

	const code = typeof payload.code === 'string' ? payload.code : defaultCode(status);
	const message = typeof payload.message === 'string' ? payload.message : DEFAULT_MESSAGE;

	return new ApiError(status, code, message, payload.details);
}
```

import { HttpResponse, http, delay } from 'msw';
import { db, type MockUser } from './db';

const BASE = '*/api/users';

export const handlers = [
	http.get(BASE, async ({ request }) => {
		await delay(300);

		const url = new URL(request.url);
		const page = Number(url.searchParams.get('page') ?? '1');
		const pageSize = Number(url.searchParams.get('pageSize') ?? '10');
		const search = (url.searchParams.get('search') ?? '').toLowerCase();
		const role = url.searchParams.get('role');
		const sort = url.searchParams.get('sort');

		let items = db.users.filter((user) => {
			const matchSearch =
				!search || user.name.toLowerCase().includes(search) || user.email.toLowerCase().includes(search);
			const matchRole = !role || role === 'all' || user.role === role;
			return matchSearch && matchRole;
		});

		if (sort === 'name_asc') items = [...items].sort((a, b) => a.name.localeCompare(b.name));
		if (sort === 'name_desc') items = [...items].sort((a, b) => b.name.localeCompare(a.name));

		const total = items.length;
		const start = (page - 1) * pageSize;

		return HttpResponse.json({
			items: items.slice(start, start + pageSize),
			total,
			page,
			pageSize,
		});
	}),

	http.post(BASE, async ({ request }) => {
		await delay(300);
		const body = (await request.json()) as Omit<MockUser, 'id' | 'createdAt'>;

		if (db.users.some((user) => user.email === body.email)) {
			return HttpResponse.json(
				{
					code: 'VALIDATION_FAILED',
					message: 'Dữ liệu không hợp lệ',
					details: { email: 'Email đã tồn tại' },
				},
				{ status: 422 },
			);
		}

		const created: MockUser = {
			...body,
			id: String(Date.now()),
			createdAt: new Date().toISOString(),
		};
		db.users = [created, ...db.users];
		return HttpResponse.json(created, { status: 201 });
	}),

	http.patch(`${BASE}/:id`, async ({ request, params }) => {
		await delay(300);
		const body = (await request.json()) as Partial<MockUser>;
		const index = db.users.findIndex((user) => user.id === params.id);

		if (index === -1) {
			return HttpResponse.json({ code: 'NOT_FOUND', message: 'Không tìm thấy người dùng' }, { status: 404 });
		}

		db.users[index] = { ...db.users[index], ...body };
		return HttpResponse.json(db.users[index]);
	}),

	http.delete(`${BASE}/:id`, async ({ params }) => {
		await delay(300);
		const exists = db.users.some((user) => user.id === params.id);

		if (!exists) {
			return HttpResponse.json({ code: 'NOT_FOUND', message: 'Không tìm thấy người dùng' }, { status: 404 });
		}

		db.users = db.users.filter((user) => user.id !== params.id);
		return new HttpResponse(null, { status: 204 });
	}),

import { QueryCache, QueryClient } from '@tanstack/react-query';
import { toast } from 'sonner';
import { ApiError } from '@/lib/apiError';

function makeQueryClient() {
	return new QueryClient({
		queryCache: new QueryCache({
			onError: (error, query) => {
				// Query nào tự xử lý lỗi thì đặt meta.skipGlobalError = true
				if (query.meta?.skipGlobalError) return;

				const message = error instanceof ApiError ? error.message : 'Không tải được dữ liệu, vui lòng thử lại.';
				toast.error(message);
			},
		}),
		defaultOptions: {
			queries: {
				staleTime: 60 * 1000,
				retry: (failureCount, error) => {
					// Không retry lỗi 4xx — retry cũng vô ích
					if (error instanceof ApiError && error.status < 500) return false;
					return failureCount < 2;
				},
			},
		},
	});
}

let browserQueryClient: QueryClient | undefined;

	setToken: (token: string | null) => void;

export function useHydrateStore(): boolean {
	const [hydrated, setHydrated] = useState(false);

'use client';

import { useEffect, useState } from 'react';

/** Trả về `value` sau khi nó ngừng thay đổi trong `delay` mili giây. */
export function useDebounce<T>(value: T, delay: number): T {
	const [debounced, setDebounced] = useState(value);

	useEffect(() => {
		const timer = setTimeout(() => setDebounced(value), delay);
		return () => clearTimeout(timer);
	}, [value, delay]);

	return debounced;
}
```

```ts
'use client';

import { useEffect, useState } from 'react';

/**
 * Theo dõi một media query. Trả false ở lần render đầu trên server
 * để server và client khớp nhau, sau đó cập nhật giá trị thật ở client.
 */
export function useMediaQuery(query: string): boolean {
	const [matches, setMatches] = useState(false);

	useEffect(() => {
		const list = window.matchMedia(query);
		setMatches(list.matches);

		const onChange = (event: MediaQueryListEvent) => setMatches(event.matches);
		list.addEventListener('change', onChange);

		return () => list.removeEventListener('change', onChange);
	}, [query]);

	return matches;
}
```

import { http } from '@/lib/http';
import type { User, UserFormValues, UserListParams, UserListResponse } from '../schema';

export const userApi = {
	list(params: UserListParams): Promise<UserListResponse> {
		const searchParams: Record<string, string> = {
			page: String(params.page),
			pageSize: String(params.pageSize),
		};
		if (params.search) searchParams.search = params.search;
		if (params.role && params.role !== 'all') searchParams.role = params.role;
		if (params.sort) searchParams.sort = params.sort;

		return http.get('users', { searchParams }).json<UserListResponse>();
	},

	create(values: UserFormValues): Promise<User> {
		return http.post('users', { json: values }).json<User>();
	},

	update(id: string, values: Partial<UserFormValues>): Promise<User> {
		return http.patch(`users/${id}`, { json: values }).json<User>();
	},

	remove(id: string): Promise<void> {
		return http.delete(`users/${id}`).json<void>();
	},
};

					void setPage(1);

							Array.from({ length: pageSize }).map((_, index) => (

										<TableCell key={String(column.id ?? Math.random())}>
											<Skeleton className="h-5 w-full" />

		if (!open) return;

			onOpenChange(false);
		} catch (error) {
			// Lỗi validate từ server: map ngược vào đúng field
			if (error instanceof ApiError && error.status === 422) {
				const details = error.details as Record<string, string> | undefined;
				if (details) {
					for (const [field, message] of Object.entries(details)) {
						form.setError(field as keyof UserFormValues, { type: 'server', message });
					}
					return;
				}
			}
			toast.error(error instanceof ApiError ? error.message : 'Lưu không thành công');
		}
	}

	const isSubmitting = createUser.isPending || updateUser.isPending;

	return (
		<Dialog open={open} onOpenChange={onOpenChange}>
			<DialogContent>
				<DialogHeader>
					<DialogTitle>{isEdit ? 'Sửa người dùng' : 'Thêm người dùng'}</DialogTitle>
					<DialogDescription>
						{isEdit ? 'Cập nhật thông tin người dùng.' : 'Nhập thông tin người dùng mới.'}
					</DialogDescription>
				</DialogHeader>

				<Form {...form}>
					<form onSubmit={form.handleSubmit(onSubmit)} className="space-y-4">
						<FormField
							control={form.control}
							name="name"
							render={({ field }) => (
								<FormItem>
									<FormLabel>Họ tên</FormLabel>
									<FormControl>
										<Input placeholder="Nguyễn Văn An" {...field} />
									</FormControl>
									<FormMessage />
								</FormItem>
							)}
						/>

						<FormField
							control={form.control}
							name="email"
							render={({ field }) => (
								<FormItem>
									<FormLabel>Email</FormLabel>
									<FormControl>
										<Input type="email" placeholder="an@example.test" {...field} />
									</FormControl>
									<FormMessage />
								</FormItem>
							)}
						/>

						<FormField
							control={form.control}
							name="role"
							render={({ field }) => (
								<FormItem>
									<FormLabel>Vai trò</FormLabel>
									<Select value={field.value} onValueChange={field.onChange}>
										<FormControl>
											<SelectTrigger>
												<SelectValue />
											</SelectTrigger>
										</FormControl>
										<SelectContent>
											<SelectItem value="admin">Quản trị</SelectItem>
											<SelectItem value="member">Thành viên</SelectItem>
										</SelectContent>
									</Select>
									<FormMessage />
								</FormItem>
							)}
						/>

						<DialogFooter>
							<Button type="button" variant="outline" onClick={() => onOpenChange(false)}>
								Huỷ
							</Button>
							<Button type="submit" disabled={isSubmitting}>
								{isSubmitting ? 'Đang lưu...' : 'Lưu'}
							</Button>
						</DialogFooter>
					</form>
				</Form>
			</DialogContent>
		</Dialog>
	);
}

```ts
'use client';

import { useInfiniteQuery, type QueryKey } from '@tanstack/react-query';
import { useEffect } from 'react';
import { useInView } from 'react-intersection-observer';

type Options<TItem> = {
	queryKey: QueryKey;
	queryFn: (page: number) => Promise<{ items: TItem[]; total: number }>;
	pageSize: number;
};

/**
 * Gộp useInfiniteQuery với IntersectionObserver: gắn `sentinelRef` vào
 * một phần tử cuối danh sách, khi nó lọt vào viewport thì tự tải trang tiếp.
 */
export function useInfiniteScroll<TItem>({ queryKey, queryFn, pageSize }: Options<TItem>) {
	const query = useInfiniteQuery({
		queryKey,
		queryFn: ({ pageParam }) => queryFn(pageParam),
		initialPageParam: 1,
		getNextPageParam: (lastPage, allPages) => {
			const loaded = allPages.reduce((sum, page) => sum + page.items.length, 0);
			return loaded < lastPage.total ? allPages.length + 1 : undefined;
		},
	});

	const { ref: sentinelRef, inView } = useInView({ rootMargin: '200px' });

	const { fetchNextPage, hasNextPage, isFetchingNextPage } = query;

	useEffect(() => {
		if (inView && hasNextPage && !isFetchingNextPage) {
			void fetchNextPage();
		}
	}, [inView, hasNextPage, isFetchingNextPage, fetchNextPage]);

	const items = query.data?.pages.flatMap((page) => page.items) ?? [];

	return {
		items,
		sentinelRef,
		isPending: query.isPending,
		isError: query.isError,
		isFetchingNextPage,
		hasNextPage: Boolean(hasNextPage),
		total: query.data?.pages[0]?.total ?? 0,
		pageSize,
	};
}
```

Guard `hasNextPage && !isFetchingNextPage` là bắt buộc — thiếu nó thì sentinel còn trong viewport sẽ gọi `fetchNextPage` liên tục.

`rootMargin: '200px'` cho phép tải trước khi người dùng cuộn tới đáy, cảm giác mượt hơn.
